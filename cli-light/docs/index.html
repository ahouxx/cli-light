功能一览

进程自动检测

每 3 秒扫描 claude.exe / kimi-cli.exe / opencode.exe，实时统计运行实例数

Hook 状态上报

通过各 CLI 原生 Hook 机制，任务状态实时推送到指示灯

一键安装

install.ps1 自动配置 Claude Code / Kimi Code / OpenCode 的 Hook，生成桌面快捷方式

多实例追踪

以进程 PID 区分 Agent，多个 CLI 实例互不干扰

拖拽 + 吸附

自由拖拽窗口到任意位置，靠近屏幕边缘自动吸附

位置记忆

退出时记住窗口位置和所有设置，重启后自动恢复

深色 / 透明主题

右键切换：深色经典风格 或 背景完全透明仅显示灯和边框

5 种配色方案

默认 / 海洋 / 森林 / 霓虹 / 琥珀，即时切换灯色

八档缩放

75% ~ 1000%，灯、文字、边框等比缩放

横向 / 竖向布局

右键菜单切换灯的排列方向，适应不同屏幕位置

显示边框

可选在每个灯周围显示方形边框，视觉更清晰

重复启动保护

检测到已有实例运行时弹窗询问，可选择启动新实例或取消

干净卸载

install.ps1 -Uninstall 一键移除 Hook、快捷方式、进程和状态文件

安装

# 1. 启动指示灯
pythonw cli_light.py

# 2. 安装 Hook（一次性）
powershell -ExecutionPolicy Bypass -File install.ps1

# 3. 重启 CLI 终端，开始使用

Hook 生命周期

UserPromptSubmit ──→ 🟠 橙灯（running）
     │
     ▼ 需要授权
PermissionRequest ──→ 🔴 红灯（needs_input）
     │
     ▼ 授权通过
PostToolUse ──→ 🟠 橙灯（running）
     │
     ▼ 任务完成
Stop ──→ 🟢 绿灯（done）

操作

  • 拖拽 —— 移动窗口
  • 双击 —— 紧急关闭
  • 右键 —— 菜单：置顶 / 显示边框 / 主题 / 配色 / 缩放 / 样式（横向/竖向） / 退出
  • 边缘吸附 —— 拖到屏幕边缘自动对齐

环境要求

  • Windows 10 / 11
  • Python 3.10+（纯标准库，零外部依赖）
